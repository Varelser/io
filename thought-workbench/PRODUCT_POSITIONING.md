# Product Positioning

最終更新: 2026-03-22

## Short

4D Thinking OS は、ノート保存よりも「思考の配置・比較・再編集」を中心にしたローカル保存型 thinking workbench です。

## README Version

4D Thinking OS は、3D / 2D 空間、時間軸、複数ペイン、分岐比較、saved view をまとめて扱うローカル PKM / thinking workbench です。未整理の断片を早く置き、後から構造化・比較・再解釈できることを重視しています。

## Sales Page Version

4D Thinking OS は、メモを「保存する」ための道具ではなく、考えを「配置して、見比べて、育てる」ための道具です。3D / 2D 空間、時間比較、分岐 sandbox、layout preset、bundle、review 支援を 1 つのワークスペースで扱えます。

## Difference From Other Tools

### vs Notion

- 文書中心ではなく、空間配置と比較中心
- 未整理断片の扱いを重視

### vs Obsidian

- リンク保存より、空間・時間・分岐の編集を重視
- vault export/import はあるが、本体は思考操作が中心

### vs XMind / mind map tools

- 単発マップではなく、保存・review・timeline・branch を同居させる

### vs task tools

- task だけでなく、仮説・観察・反証・資料も同じ graph で扱う
